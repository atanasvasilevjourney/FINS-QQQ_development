# Range Breakout EA - Installation Guide

## Step-by-Step Installation

### Step 1: Download and Prepare Files
1. Download the RangeBreakoutEA.mq5 file
2. Ensure you have MetaTrader 5 installed
3. Make sure you have a trading account with a broker

### Step 2: Install the Expert Advisor
1. Open MetaTrader 5
2. Press F4 or go to Tools → MetaQuotes Language Editor
3. In MetaEditor, go to File → Open
4. Navigate to your MT5 data folder (usually: C:\Users\[Username]\AppData\Roaming\MetaQuotes\Terminal\[TerminalID]\MQL5\Experts)
5. Copy the RangeBreakoutEA.mq5 file to this folder
6. In MetaEditor, open the RangeBreakoutEA.mq5 file
7. Press F7 or click the Compile button
8. Check for any compilation errors and fix them if necessary

### Step 3: Attach to Chart
1. In MetaTrader 5, go to the Navigator panel (View → Navigator)
2. Expand the Expert Advisors folder
3. Find "RangeBreakoutEA" in the list
4. Drag and drop it onto your desired chart
5. A settings window will appear

### Step 4: Configure Settings
1. **Range Settings**:
   - Range Start Hour: 3
   - Range Start Minute: 0
   - Range End Hour: 6
   - Range End Minute: 0

2. **Trading Settings**:
   - Trading End Hour: 18
   - Trading End Minute: 0
   - Risk Amount: 50.0 (adjust based on your account)
   - Magic Number: 12345 (or any unique number)

3. **Advanced Settings**:
   - Max Trades Per Day: 1
   - Allow Multiple Breakouts: false
   - Min Range Size: 0.0001
   - Show Range Visualization: true

4. Click OK to apply settings

### Step 5: Enable Auto Trading
1. In MetaTrader 5, go to Tools → Options
2. Go to the Expert Advisors tab
3. Check "Allow automated trading"
4. Check "Allow DLL imports" (if needed)
5. Click OK

### Step 6: Verify Installation
1. Check that the EA appears in the chart's Expert Advisors list
2. Look for the smiley face icon next to the EA name (indicates it's running)
3. Check the Experts tab in the Terminal for any error messages
4. Wait for the range period to see if the yellow rectangle appears

## Troubleshooting

### Common Issues and Solutions

#### EA Not Appearing in Navigator
- **Solution**: Make sure the file is in the correct Experts folder
- **Check**: File path should end with \MQL5\Experts\RangeBreakoutEA.mq5

#### Compilation Errors
- **Solution**: Check for syntax errors in the code
- **Common Issues**: Missing semicolons, incorrect variable names
- **Fix**: Review error messages and correct the code

#### EA Not Running (No Smiley Face)
- **Solution**: Check if Auto Trading is enabled
- **Check**: Tools → Options → Expert Advisors → Allow automated trading

#### No Range Rectangle Appearing
- **Solution**: Wait for the range period (3:00-6:00 AM by default)
- **Check**: Ensure market is open during range period
- **Verify**: Check if Show Range Visualization is set to true

#### EA Not Opening Positions
- **Solution**: Check account balance and margin
- **Check**: Ensure sufficient funds for calculated position size
- **Verify**: Check if range size meets minimum requirements

#### Positions Not Closing
- **Solution**: Check trading end time settings
- **Verify**: Ensure EA is still running
- **Check**: Look for error messages in Experts tab

### Testing the EA

#### Demo Account Testing
1. Open a demo account with your broker
2. Install the EA on demo account
3. Use small risk amounts for testing
4. Monitor for at least 1 week before live trading

#### Backtesting
1. Go to Tools → Strategy Tester
2. Select RangeBreakoutEA from the Expert Advisor dropdown
3. Choose your symbol and timeframe
4. Set the testing period (recommend at least 6 months)
5. Click Start to begin backtesting
6. Review the results and statistics
